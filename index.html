Workflow Automation

Workflow automation rules help you automate repetitive tasks when vehicles move through different recon stages.

Benefits of Automation

  • Automatically assign technicians when a vehicle enters a specific stage
  • Set due dates based on recon stage
  • Send notifications to stakeholders
  • Ensure consistent processes for every vehicle
  • Reduce manual work and human error

How It Works In Practice

When a vehicle enters a stage that has rules defined, those rules will automatically execute:

  1. Vehicle enters "Inspection" stage
  2. System checks for rules defined for "Inspection"
  3. Rules are applied:
    • Assign technician "Alex Johnson"
    • Set due date to 2 days from now
    • Send notification to manager
